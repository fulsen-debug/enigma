1) Connect Wallet

Open Dashboard, click Connect Phantom, then wait for login confirmation.

Note: You do not need to enter Helius/Jupiter/admin API keys as a user.

Scanner access: wallets must hold at least 500,000 KOBX. Daily scan tiers are 2/day for 500K+ and 5/day for 3M+.

2) Scan a Token

Use Scan Random Token for discovery or paste one token into Manual token scan.

Each scan reviews one token at a time and builds a full report before any execution decision.

3) Subscribe to Premium

Go to Premium Checkout.

  1. Choose tier.
  2. Send SOL to the shown payment address.
  3. Paste transaction signature and verify.
  4. Reconnect wallet to refresh plan badge to PRO.

4) Trading Console: Test Mode First

  1. Open Trading Console (Paper Agent).
  2. Set one Agent Token.
  3. Set Paper Budget, Test Amount Per Trade, and Check Every (sec).
  4. Click Run Test and let the paper loop monitor the token automatically.
  5. Review entries/exits in Trade Activity (Buy/Sell Visual) and table results.

Note: The agent is deterministic and paper-only in this first release. Each paper run auto-stops after 8 minutes.

5) First Release Boundary

This release ships:

  • Scanner Workspace for quick one-token analysis
  • Agent Workspace for autonomous paper trading on one selected token

This release does not ship public live execution.
