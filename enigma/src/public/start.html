Fast onboarding for non-technical traders

1) Connect Wallet

Open Dashboard, click Connect Phantom, then wait for login confirmation.

Note: You do not need to enter Helius/Jupiter/admin API keys as a user.

2) Build Your Watchlist

Add up to 5 mint addresses and click Save Watchlist.

Run Scan Now to validate token quality before any execution.

3) Subscribe to Premium

Go to Premium Checkout.

  1. Choose tier.
  2. Send SOL to the shown payment address.
  3. Paste transaction signature and verify.
  4. Reconnect wallet to refresh plan badge to PRO.

4) Trading Console: Test Mode First

  1. Open Trading Console (Test + Live Agent).
  2. Set Agent Token Mint (single token for both Test and Live agent loops).
  3. In Test Mode (Paper), click Apply $25 Beginner Preset.
  4. Pick a tempo preset (1m Scalp, 5m Momentum, or 15m Cycle) if you want faster accumulation loops.
  5. Optional for faster entries: enable Allow CAUTION entries under Advanced Test Rules.
  6. Click Run Test Mode and let it run for several cycles.
  7. Review entries/exits in Trade Activity (Buy/Sell Visual) and table results.

Note: Watchlist is scanner-only.

5) Live Agent (PRO for Real Execution)

  1. Set Trade Size USD, Max Open Positions, and Entry Check Interval.
  2. Layered same-token entries are allowed until Max Open Positions is reached.
  3. Choose an exit preset (Scalp, Balanced, Swing).
  4. Click Run Live Agent after test mode is stable.
  5. Use Stop Live Agent any time to halt autonomous ticks.

Live execution requires PRO plan and backend flag ENIGMA_EXECUTION_ENABLED=1.
